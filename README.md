# SkillSight AI

SkillSight AI is a multimodal RAG-based Student and Career Assistant that helps users interact with PDFs, resumes, notes, research papers, screenshots, and scanned documents.

## Project Focus

- AI Engineering
- Data Science
- Generative AI
- Retrieval-Augmented Generation
- Vector Databases
- Multimodal Document Understanding

## Core Idea

Users can upload documents and ask questions. The assistant retrieves relevant content from uploaded files and generates source-grounded answers using Gemini.
